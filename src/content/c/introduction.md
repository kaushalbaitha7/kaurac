# Introduction to C Programming

## What is C?

C is a general-purpose, procedural programming language developed by **Dennis Ritchie** at Bell Laboratories in 1972.

It is one of the most influential programming languages and serves as the foundation for many modern languages like C++, Java, C#, and others.
