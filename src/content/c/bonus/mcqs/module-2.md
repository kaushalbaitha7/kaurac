# Question 1

Which operator is used for addition?

- A. +
- B. -
- C. *
- D. /

✅ Correct Answer

**A. +**

---

# Question 2

Which operator is used to compare two values?

- A. =
- B. ==
- C. +=
- D. :=

✅ Correct Answer

**B. ==**

# Question 3

# Question 14
