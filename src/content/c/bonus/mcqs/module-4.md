# Question 1

A pointer is:

- A. A keyword
- B. A variable that stores the address of another variable
- C. A function
- D. A constant

✅ Correct Answer

**B. A variable that stores the address of another variable**

---

# Question 2

Which operator is used to obtain the address of a variable?

- A. *
- B. &
- C. %
- D. #

✅ Correct Answer

**B. &**

---

# Question 3

Which operator is used to access the value stored at a memory address?

- A. &
- B. *
- C. %
- D. #

✅ Correct Answer

**B. \***

---

# Question 4

Which declaration creates a pointer to an integer?

- A. int ptr;
- B. int *ptr;
- C. pointer int;
- D. ptr int;

✅ Correct Answer

**B. int *ptr;**

---

# Question 5

Which of the following is a NULL pointer?

```c
int *ptr = NULL;
```

- A. Invalid
- B. Pointer to zero
- C. Pointer that points to no valid memory
- D. Integer variable

✅ Correct Answer

**C. Pointer that points to no valid memory**

---

# Question 6

Which operator is used with pointers to access values?

- A. %
- B. *
- C. &
- D. +

✅ Correct Answer

**B. \***

---

# Question 7

Pointer arithmetic is generally performed using:

- A. +
- B. -
- C. ++ / --
- D. All of the above

✅ Correct Answer

**D. All of the above**

---

# Question 8

A double pointer stores:

# Question 13

- A. It cannot store data.

# Question 16

- A. malloc()
- B. realloc()
- C. calloc()

- A. calloc()
- B. realloc()
- C. malloc()

# Question 18

Which function releases dynamically allocated memory?

- A. delete()
- B. release()
- C. free()

# Question 19

Failing to call `free()` after `malloc()` may cause:

- A. Compilation Error
- B. Syntax Error
- C. Memory Leak
- D. Runtime Crash Always

✅ Correct Answer

**C. Memory Leak**

---

# Question 20

What does `malloc()` return on failure?

- A. 0
- B. NULL
- C. -1

# Question 21

# Question 22

A dangling pointer is:

- A. A pointer pointing to valid memory
- B. A pointer that has been freed but is still used
- C. A NULL pointer
- D. A constant pointer

✅ Correct Answer

**B. A pointer that has been freed but is still used**

---

# Question 23

Which statement about structures is TRUE?

- A. Structures cannot contain arrays.
- B. Structures can contain variables of different data types.
- C. Structures cannot contain pointers.
- D. Structures are only used in C++.

✅ Correct Answer

**B. Structures can contain variables of different data types.**

---

# Question 24

Which statement about unions is TRUE?

- A. Every member has separate memory.
- B. All members are active simultaneously.
- C. Only one member's value is valid at a time.
- D. Unions cannot contain integers.

✅ Correct Answer

**C. Only one member's value is valid at a time.**

---

# Question 25

Dynamic Memory Allocation allows memory to be allocated:

- A. Before compilation
- B. During program execution
- C. During installation
- D. During linking

✅ Correct Answer

**B. During program execution**
