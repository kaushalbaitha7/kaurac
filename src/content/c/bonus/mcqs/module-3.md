# Question 1

A function is mainly used to:

- A. Increase program size
- B. Reduce code duplication
- C. Slow down execution
- D. Store data

✅ Correct Answer

**B. Reduce code duplication**

---

# Question 2

Which keyword is used to return a value from a function?

- A. break
- B. exit
- C. return
- D. continue

✅ Correct Answer

**C. return**

---

# Question 3

Which function is called automatically when a C program starts?

- A. start()
- B. begin()
- C. execute()
- D. main()

✅ Correct Answer

**D. main()**

# Question 4

Which of the following is a valid function declaration?

- A. int sum(int, int);
- B. function sum();

- D. declare sum()

**A. int sum(int, int);**

# Question 5
