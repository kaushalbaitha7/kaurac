# Interview Tips

Before attending a technical interview:

- Explain concepts confidently.
- Use simple language and practical examples.
- If you don't know an answer, explain your thought process.
- Be honest instead of guessing.
- Revise pointers, arrays, structures, functions, and file handling thoroughly.
- Practice writing code on paper, not just in an IDE.
- Maintain eye contact and communicate clearly.
