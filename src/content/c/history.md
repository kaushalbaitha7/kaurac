# History of C Programming

## Overview

The C programming language was developed by **Dennis Ritchie** in **1972** at **Bell Laboratories (Bell Labs)**.

It was created to develop the UNIX operating system and became one of the most successful programming languages in history.

---

```

      C (1972)
      ↓

```

---

## Dennis Ritchie

Dennis Ritchie is known as the **Father of C Programming**.

He developed C while working at Bell Labs.

His work on C and UNIX completely changed software engineering.

---

## Why was C Created?

Earlier programming languages had several limitations.

C was designed to provide:

- High Performance
- Portability
- Low-level memory access
- Structured programming
- Easy compiler development
